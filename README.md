# Digital Twin (ABM Project)

Digital Twin is an advanced Agent-Based Modeling (ABM) framework developed by Bastard Software.

## 🚀 Usage & Research
This software is provided for free for **personal use** and **academic research**. We encourage scientists and hobbyists to explore, modify, and build upon this code.
